# AI Color Palette & Prompt Generator

This is a modern web application for generating color palettes and AI-ready design prompts.

## Features

### 1. Color Palette Generator
- 10 brand-inspired styles: Nothing, Claude, Vercel, Linear, Stripe, Notion, Spotify, Discord, Figma, Glass
- Separate Dark/Light mode toggle
- 6 colors per palette: primary, secondary, accent, background, surface, text
- Auto-generated palette names
- Click-to-copy hex codes

### 2. Live UI Preview
- Compact mini preview showing navbar, cards, buttons, stats
- Dynamic styling based on selected palette and mode

### 3. Vibe Mode (Simplified)
- 4 simple vibes: minimal, bold, playful, techy
- Generates concise AI prompts

### 4. Dev Export
- CSS Variables, Tailwind config, JSON formats
- One-click copy to clipboard

### 5. Saved Palettes
- Requires authentication to save
- Load/delete saved palettes
